wasd or arrow keys - up/left/down/right

q - thumbs up
e - da bird

Rules of janken:
First is rock. Then you shoot. (on pon)
If you tie, you go again (on desho)
If someone wins the first round, you go to achimuite (look over there)
Pick a direction. Whoever won the first part wins if the directions match.
either way, you start over.

Make sure you throw the right thing, and don't change your move once you've picked it!
